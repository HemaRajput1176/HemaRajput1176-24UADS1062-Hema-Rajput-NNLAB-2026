First, I installed all required libraries like PyTorch, Torchvision, NumPy, Matplotlib, Seaborn, Pillow, and Scikit-learn Then I securely connected Colab to Kaggle using API authentication and downloaded the dataset containing 5,856 chest X-ray images divided into NORMAL and PNEUMONIA classes
I loaded the images using PyTorch DataLoaders and applied data augmentation to training images to avoid overfitting All images were resized to 224×224 pixels to match the input size of ResNet I used ResNet50 pretrained on ImageNet, froze most layers, and added my own classification head Only 4.5% parameters were trainable
To handle class imbalance I used weighted cross-entropy loss, along with the Adam optimizer and a learning rate scheduler The model was trained for up to 20 epochs with early stopping and achieved the best validation accuracy of 93.75% at epoch 6
On the unseen test set of 624 images, the model achieved 89.42% accuracy, 96.67% sensitivity, 77.35% specificity, 91.95% F1 score, and AUC-ROC of 0.9587, which is clinically strong. I also plotted confusion matrix, ROC curve, loss and accuracy graphs, and showed sample predictions
I saved the final model and medical metrics, demonstrated inference on sample images, and implemented Grad-CAM explainability to highlight which regions of the X-ray the model used for prediction. Finally, I created an image upload feature in Colab so users can upload their own X-ray and get predictions with a medical note
